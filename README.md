Telegram-бот с RPG механикой, где игроки сражаются с врагами в лесу, прокачивают характеристики и покупают снаряжение.

## Описание

Игровой бот для Telegram с системой боев, прокачки персонажа и магазином. Игроки проходят через лес, сражаясь с различными противниками от гоблинов до боссов, улучшая свои навыки и экипировку.

## Основные возможности

- Регистрация и создание персонажа
- Система характеристик (HP, сила, ловкость, удача, броня)
- Тренировки для прокачки навыков
- Битвы с противниками (гоблины, рыцари, ивент-боссы)
- Магазин с броней разных уровней
- Система валют (обычные и ивент монеты)
- Аптечки и восстановление HP

## Технологии

- Python 3
- aiogram (Telegram Bot API)
- SQLite (база данных)

## Установка

1. Установите зависимости:
```bash
pip install aiogram
```

2. Укажите токен бота в `config.py`:
```python
TOKEN = "ваш_токен"
```

3. Запустите бота:
```bash
python main.py
```

## Структура проекта

- `main.py` - основная логика бота и обработчики
- `base.py` - класс для работы с SQLite БД
- `config.py` - конфигурация (токен бота)
- `db.db` - база данных SQLite
- `Статусы.txt` - описание статусов пользователя

## Игровая механика

Игроки выбирают направление атаки/уклонения (лево/центр/право), успех определяется случайным образом. За победы начисляются награды: характеристики, монеты и опыт.
